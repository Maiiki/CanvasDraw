Handles keyboard input.

Parameters
keyValue of the pressed/released key.
isPressedIndicates if the key is pressed or released.

Keyboard inputs handled:

  • C | Draw a circle
  • S | Draw a rectangle
  • T | Draw a Triangle
  • F | Draw a Fly Eye
  • I | Alternate between isometric an normal view
  • X | Delete all shapes
  • Z | Delete the last shape
